Клонирование Проекта с Гитхаба
```commandline
git clone <ссылка на репозиторий>
```
Добавление файлов в наш коммит
```commandline
git add .
```
Отмена Добавление
```commandline
git reset .
```
Проверка Статуса
```commandline
git status
```
Создание Коммита с его названием
```commandline
git commit -m "Название"
```
Сохранения нашего коммита в репозитории
```commandline
git push
```
